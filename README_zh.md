# LRC 视频字幕同步器

[English](README.md) | 中文

一个基于浏览器的视频字幕同步工具，可以为视频分配字幕，最终导出标准的 LRC 字幕文件。

## 🎯 核心功能

- **视频切割**：在任意时间点切割视频为多个段落
- **字幕分配**：拖拽式字幕分配到对应时间段
- **时间轴可视化**：直观的时间轴界面，悬停查看完整字幕
- **LRC 导出**：生成标准格式的 LRC 字幕文件
- **批量导入**：支持批量粘贴字幕文本，自动分句

## 🚀 快速开始

### 开发模式
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

### Chrome 插件模式
1. 运行 `npm run build` 构建项目
2. 打开 Chrome 扩展程序页面 (chrome://extensions/)
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"，选择 `dist` 目录
5. 或者选择"打包扩展程序"，选择 `dist` 目录，打包成 crx 文件
6. 开始使用插件

### 使用流程
1. 导入视频文件
2. 播放视频，按需切割分段
3. 准备字幕文本（单条或批量）
4. 拖拽字幕到对应时间段
5. 导出 LRC 文件

## 🛠 技术栈

- React + TypeScript
- Tailwind CSS
- Vite 构建工具
- 纯前端实现，无需后端服务器

## 📦 项目特点

- **纯客户端**：所有处理在浏览器中完成，无需上传视频
- **拖拽交互**：直观的拖拽式字幕分配
- **快捷键支持**：空格播放/暂停，M 键切割

## 🔧 环境要求

- Node.js 16+
- 现代浏览器（Chrome 90+ / Firefox 88+ / Safari 14+）
